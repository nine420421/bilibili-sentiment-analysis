# B站视频评论情感分析系统

一个基于Streamlit的B站评论情感分析Web应用。

## 功能特点

- 📊 评论数据可视化分析
- 🎭 情感倾向分析（积极/消极/中性）
- ☁️ 动态词云生成
- 💬 评论详情浏览和筛选
- 📈 多种图表展示

## 技术栈

- Python
- Streamlit
- Pandas
- Matplotlib
- WordCloud
- Jieba
- Plotly

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
